# Superpowers

This project uses the **Superpowers** engineering workflow.

## Instructions
- Before starting any engineering task, check the global Superpowers library at `~/.gemini/extensions/superpowers/skills`.
- **Mandatory Phases**: 
  1. **Brainstorming**: Refine the spec through questions.
  2. **Writing Plans**: Create a detailed plan with atomic tasks and verification.
  3. **TDD**: Write a failing test first for every bug or feature.
- Follow the Systematic Debugging skill for any complex errors.

## Traceability
- Follow the local `.agent/rules/traceability-protocol.md` for all commit messages and step logging.
